Text / search input with optional leading icon — the AI-chat composer and search fields.

```jsx
<TextInput placeholder="輸入問題，例如：東區今年平均單價是多少？" />
<TextInput size="sm" icon={<span>⌕</span>} placeholder="搜尋建案…" />
```

Sizes `lg` (40px, default) and `sm` (32px). Wrap width with `wrapStyle={{maxWidth: 420}}`.
