Data panel вҖ” the rounded card wrapping tables and charts. Brass tick + title + optional count, with right-aligned `actions`.

```jsx
<Panel title="иЎҢж”ҝеҚҖжҺ’иЎҢ" count={37} actions={<Button size="sm" variant="ghost">еҢҜеҮә</Button>}>
  вҖҰchart or tableвҖҰ
</Panel>
<Panel title="йЎһеһӢзөұиЁҲ" flush>{/* edge-to-edge table */}</Panel>
```

Use `flush` for tables that should reach the card edges. Omit `title` for a plain card.
