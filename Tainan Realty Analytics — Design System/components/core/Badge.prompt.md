Status / category pill — semantic tone colours for property types and states; a quiet `count` variant for tallies.

```jsx
<Badge tone="positive">成屋</Badge>
<Badge tone="info">預售</Badge>
<Badge tone="brass" dot pulse>最新 115年6月</Badge>
<Badge variant="count">37</Badge>
```

Tones: `brass` (default), `clay`, `positive`, `info`, `warning`, `negative`, `neutral`. Use `dot`/`pulse` for live indicators, `variant="count"` for row counts.
